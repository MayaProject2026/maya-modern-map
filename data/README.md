# Data

This folder contains the geospatial data for the Maya Modern Map project.

## Files

- `maya_sites.geojson` — Converted Maya archaeological site data (approximately 5,200 named sites). Generated by `scripts/convert_sites.py`.

## Source Data

The source shapefile (EAAMS, compiled by Walter Witschey) is not redistributed in this repository. To regenerate `maya_sites.geojson`, obtain the source shapefile and place the following files in `data/raw/`:

- `EAAMS.shp`
- `EAAMS.dbf`
- `EAAMS.shx`
- `EAAMS.prj`

Then run the conversion script from the repository root:
python scripts/convert_sites.py
## Coordinate Reference System

All geographic data is in WGS84 (EPSG:4326) with decimal-degree longitude and latitude values, conforming to RFC 7946 (GeoJSON).
