Maya Modern Map

A modernised reimplementation of mayamap.org

Status: Project under active development. The interactive map will appear here once implementation is complete.

This site hosts the working artifact for an MSc Data Science dissertation at the University of St Andrews (CS5999), supervised by Professor Lars Kotthoff.

The project modernises the existing mayamap.org interactive web map of Maya archaeological sites by replacing its OpenLayers 2 / server-side raster tile architecture with a fully client-side implementation using Leaflet.js and GeoJSON.

Source code and progress: github.com/MayaProject2026/maya-modern-map
